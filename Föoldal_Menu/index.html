Projekt Áttekintés

Ez a webalkalmazás a Web programozás-1 előadás keretében készült átfogó projektmunka. Célja a modern webes technológiák (JavaScript, React, API kezelés) gyakorlati alkalmazásának bemutatása egy egységes keretrendszerben.

Felhasznált Technológiák

JS

Vanilla JavaScript

Alapvető CRUD műveletek és objektumorientált (OOJS) grafikus megoldások.

RE

React & Axios

Modern komponensalapú UI fejlesztés, állapotkezelés (useState) és szerver oldali adatlekérés.

API

Fetch & Axios

Szerveroldali adatbázis kapcsolat (PHP/MySQL) aszinkron hívásokon keresztül.

Hogyan böngésszen a feladatok között?

  • JavaScript: Tömbalapú CRUD alkalmazás megtekintése.
  • React/SPA: Modern egyoldalas alkalmazások és játékok kipróbálása.
  • Adatbázis: Fetch API és Axios alapú éles adatkezelés szerver kapcsolattal.
  • OOJS: Grafikus, osztályalapú vizuális alkalmazás.
